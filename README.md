# Signyx-Text-Speech-to-Indian-Sign-Language
Signyx is a multimodal Indian Sign Language (ISL) translation system designed to reduce the communication gap between hearing-impaired individuals and the general public. It enables users to communicate effectively using both text and speech inputs.
The system accepts user input in the form of text or voice. If speech is provided, it is converted into text using a speech recognition module, allowing seamless interaction for users.
The input text is then processed using Natural Language Processing (NLP) techniques such as tokenization and text simplification to convert it into an ISL-compatible format.
After processing, the system maps each word to predefined sign gestures stored in the dataset. These gestures are structured and prepared for visual representation.
Finally, the output is displayed through an animated avatar that performs the corresponding sign language gestures, enabling clear and effective communication for hearing-impaired users.

Process Workflow:
Text / Speech Input → Speech-to-Text → NLP Processing → ISL Mapping → Avatar Animation → Output (Sign Language)
